# 思ってたんと違う

うまくいかなかった時の原因をまとめます。チェックリストのようにして高速に確認できるようにします。

## チェックリスト

- [x] デバック用の出力は消したか
- [x] 配列外参照
- [x] 0-based vs 1-based
- [x] 格納する変数のミスはないか 似た変数に間違って格納してないか
- [x] vectorのサイズは適切か(n, m, qの間違いを確認)
- [x] 値渡しと参照渡しを適切に選択できているか
